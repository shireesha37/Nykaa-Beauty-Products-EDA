# Nykaa Beauty Products - Exploratory Data Analysis

## Project Overview
This project presents exploratory data analysis performed on Nykaa beauty products dataset.

## Tools Used
- Python
- Pandas
- Matplotlib
- PowerPoint for presentation

## Analysis Performed
- Data cleaning
- Handling missing values
- Category-wise analysis
- Price and discount analysis
- Rating and review analysis

## Key Insights
- Identified top-rated product categories
- Observed pricing distribution trends
- Analyzed discount impact on product performance
